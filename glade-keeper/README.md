# Glade Keeper

A cozy, low-poly third-person game about tending a peaceful valley and
helping its animals. See `docs/GDD.md` for the full design doc.

## Run it
1. Install **Godot 4.3 or later** (Standard/GL Compatibility build, not
   required to be .NET/C# — this project is pure GDScript):
   https://godotengine.org/download
2. Open Godot, click **Import**, and select `glade-keeper/project.godot`.
3. Press **F5** (or the ▶ play button, top-right) to run.

## Controls
- `WASD` — move
- Mouse — look around
- `E` — help the nearest animal (walk up to one first)
- `Esc` — release/recapture the mouse cursor

## If something errors on launch
Godot will show a red error in the bottom **Output**/**Debugger** panel
with a file name and line number — copy/paste that back and it can
usually be fixed without needing to see the game running.
